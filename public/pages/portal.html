Portal Defender
Physics-based space shooter with gravity mechanics

This was a solo project built from scratch using Unity and online assets. Core mechanics were created using Euler calculations, with animations triggered by collision events. Enemies and powerups spawn randomly throughout the map. I also built the UI system displaying player information.

Rotational Mechanics & Player Controller

  • Designed and implemented the core gameplay loop using Euler-based rotational and gravity calculations.
    • Created a gravity-based navigation system that simulates orbital pull toward a central black hole.
    • Mapped vertical input to radius control and horizontal input to angle rotation using polar coordinates.
    • Added inverse-square gravity scaling for realistic orbital decay.
    • Used position clamping and angle normalization for smooth, controlled gameplay.

Enemy AI Programming

  • Built a procedural enemy spawning system with dynamic difficulty and three enemy archetypes:
    • Asteroid: Passive obstacle with randomized spawn behavior.
    • Shooter: Stationary enemy forming a defensive ring and firing projectiles.
    • Follower: Homing enemy using vector math and rotation tracking to pursue player.
  • Used random spawn coordinates and timers to ensure varied, unpredictable encounters.
  • Implemented shooter AI that halts within radius and reverses bullet direction.

Camera Programming

  • Implemented a dynamic screen shake effect using the Observer pattern.
  • Triggered shake on player damage through a static event channel.
  • Used tweening on camera position and rotation to achieve responsive shake feedback.

Particle Animation & UI/UX Programming

  • Added particle effects for key interactions like player collisions.
  • Created a modular UI system with player stats and an options menu.
  • Included settings for audio/SFX volume and resolution scaling.
